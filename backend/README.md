# Daily Dubai Lottery - Backend

This directory contains the Express.js backend for the Daily Dubai Lottery application.

## Development

To run the backend locally, ensure you have Node.js and a MySQL server running.

1.  Install dependencies: `npm install`
2.  Create a `.env` file with your database credentials and secrets. See the root `README.md` for the required format.
3.  Run the database seed script to set up tables and default users: `npm run db:seed`
4.  Start the development server: `npm run dev`

## Deployment

For full production deployment instructions, including database setup, frontend build, and Nginx configuration, please refer to the main **`README.md`** file in the project's root directory.
